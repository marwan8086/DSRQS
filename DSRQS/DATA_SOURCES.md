# Biomedical Knowledge Graph Data Sources

This project uses curated and officially licensed biomedical knowledge bases.  
All datasets are obtained through their official portals under academic or open licenses.

---

## 1. OMIM — Online Mendelian Inheritance in Man
**Official website:**  
https://omim.org/

**Download portal:**  
https://omim.org/downloads

**Data files used:**
- mim2gene.txt  
- morbidmap.txt  

**License:**  
Requires individual registration and acceptance of the OMIM data use agreement.  
OMIM data is not redistributed in this repository.
