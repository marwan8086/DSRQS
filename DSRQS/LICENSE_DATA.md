
# Data Licensing & Redistribution Policy

This repository respects all dataset licenses and redistribution restrictions.

---

### ❌ OMIM
OMIM data **cannot be redistributed**.
Users must obtain it from the official portal:
https://omim.org/downloads

This repository provides:
- loaders
- converters
- processing scripts

But **does not contain any OMIM data**.

Open for academic and research use.  
Redistribution of *processed* data (scores removed) is permitted.
